Dev Log: Marbles

May 31st, 2019:

Setting Up The Basics

This is a small project that should be fairly easy to make overall. The goal is to make this into a marble "racing" game. I'm hoping to implement a simple in-game map designer where the player can create their own maps. Right now, I have added the basics to the game. There is a timer, pause and play button, reset button, working pinwheel generator, tiny library of basic building blocks, and the marble. The marble has a little trail that emits from it as it moves. I'm working on this project now because I needed something simple to clear my mind from the Tranquility project. The other project requires me to learn so much material before I can implement it, a nice easy project like this really helps make the learning process go by easier.
