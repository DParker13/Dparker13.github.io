Dev Log: Marbles

November 20th, 2019:

Well this one didn't last too long...

My drive for this project was pretty good at the time, but of course adding some things can be extremely difficult. Turns out creating an in-game map editor is a poorly documented thing on the internet. I have looked for countless ways to implement an in-game editor but could never figure out an actual solution to my problem. I always have issues with finding a way to properly save and load the map. Figuring out how to keep track of all the player's edits is challenging and college classes are making it hard to work on any projects. I will try to revive this project in the future but have yet again moved to other projects to continue learning instead of being stuck on one issue for hours and hours and never making any progress. Once I clear my mind I'm sure I can return to this project and fix some of these issues.

September 9th, 2019:

Adding In-Game Map Editor

With this project being my main focus now, my goal is to add an in-game map editor at some point. I want to add the ability for players to drag and drop map items into the game space, and be able to design their own maps and save them. I am currently working on a way to save and load maps from file so they can be easily created and shared.

May 31st, 2019:

Setting Up The Basics

This is a small project that should be fairly easy to make overall. The goal is to make this into a marble "racing" game. I'm hoping to implement a simple in-game map designer where the player can create their own maps. Right now, I have added the basics to the game. There is a timer, pause and play button, reset button, working pinwheel generator, tiny library of basic building blocks, and the marble. The marble has a little trail that emits from it as it moves. I'm working on this project now because I needed something simple to clear my mind from the Tranquility project. The other project requires me to learn so much material before I can implement it, a nice easy project like this really helps make the learning process go by easier.
